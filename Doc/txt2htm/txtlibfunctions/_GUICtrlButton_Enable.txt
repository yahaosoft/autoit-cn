###用户自定义函数###
_GUICtrlButton_Enable

###描述###
应用或者禁用鼠标和键盘对指定按钮的输入

###语法###
#Include <GuiButton.au3>
_GUICtrlButton_Enable( $hWnd[, $fEnable = True] )

###参数###
@@ParamTable@@
$hWnd 
   控件句柄
$fEnable 
   指定按钮是否启用或禁用:
    True C 启用
    False C 禁用
@@End@@

###返回值###
@@ReturnTable@@
成功: True
失败: False
@@End@@
